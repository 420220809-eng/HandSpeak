# Sign Language Recognition API

Arabic Sign Language Recognition API using GRU and MediaPipe.

## Deployment

This API is deployed on Render.

## Endpoints

- Health: /api/health
- Model Info: /api/model-info
- Predict: /api/predict
- Docs: /api/docs

## Usage

Upload a video file to /api/predict to get sign language prediction.

See /api/docs for interactive documentation.
